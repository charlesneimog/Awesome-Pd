# add

The `add` object in Pure Data accumulates input values to a running sum, starting from a configurable initial value (0 by default). The starting sum can be set as an argument or via the `set` message. Sending a `bang` resets the accumulated sum back to the starting value. The object outputs the current accumulated value and also accepts floats to update the sum directly.
