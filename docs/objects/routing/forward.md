# forward

The `forward` object sends messages to various destinations, allowing the target to change dynamically with each message, similar to using semicolons in message boxes. It can send to any destination that a `send` object can, including arrays or the `pd` object. However, this object is considered redundant and deprecated; it's recommended to use `send` or message boxes directly instead.
