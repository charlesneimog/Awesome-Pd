# pix_video

The `pix_video` object captures live video input from various cameras (USB, FireWire, capture cards) supported by the operating system. It outputs the video stream as a `gemlist` and provides an `info` outlet for device enumeration and property management. Users can control device selection, backend, colorspace, and dimensions, making it a versatile tool for integrating live camera feeds into Pure Data patches.
