# abclib

`abclib`library is released by the CICM / MUSIDANSE (Centre de Recherches Informatique et Cr√©ation Musicale, Paris 8 University) and is the result of 20 years of research, teaching and creation in mixed music, expressed as a set of codes in Faust language. The main topics addressed are:

* spatial sound processing and synthesis using ambisonics;
* multi-channel sound processing;
* utility objects for mixed music;
